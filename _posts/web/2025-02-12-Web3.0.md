## Learning resources

### **Beginner-Friendly Resources**
- [Ethereum.org](https://ethereum.org/en/developers/) – _Very great_ for learning about Ethereum, smart contracts, and dApps.  
- [Buildspace](https://buildspace.so/) – Hands-on projects to build Web3 applications. Shut down in August 2024. :(  
- [Alchemy University](https://university.alchemy.com/) – Free courses on blockchain development and smart contracts. 

### **Online Courses**
- [Coursera: Blockchain Specialization](https://www.coursera.org/specializations/blockchain) – Covers blockchain, cryptography, and dApp development.  
- [CryptoZombies](https://cryptozombies.io/) – A fun way to learn Solidity (Ethereum smart contract language).  
- [Udemy: Ethereum & Solidity Developer Course](https://www.udemy.com/course/blockchain-developer/) – Hands-on Web3 development.  

### **Communities & Docs**
- [Solidity Documentation](https://soliditylang.org/docs/) – Official docs for writing smart contracts.  
- [Web3.js Documentation](https://web3js.readthedocs.io/) – Learn how to interact with Ethereum blockchain.  
- [Dapp University (YouTube)](https://www.youtube.com/c/DappUniversity) – Free tutorials on blockchain development.

## Solana

- [Official curriculum repository](https://github.com/solana-foundation/curriculum?tab=readme-ov-file)

- [Official tutorial](https://solana.com/learn): it contains four parts:
    - [Documentation](https://solana.com/docs): official Solana documentation and API references; a lighter doc can be found [here](https://docs.solanalabs.com/).
      - About the PoH: the blocks can be produced only sequentially (using one single CPU) but can be verified concurrently (using GPU thousands of cores), but the lengths of time consumed are proportional and is not really the _VDF_ (vérifiable delay function) popularised. Reference: a [blog]https://ackee.xyz/solana/book/latest/chapter2/proof-of-history/) and the [Solana labs' docs](https://docs.solanalabs.com/consensus/synchronization), and the 2019 paper [Verifiable Delay Functions](https://eprint.iacr.org/2018/601.pdf).
    - [Cookbook](https://solana.com/developers/cookbook): code snippets and recipes for common tasks
    - [Courses](https://learn.blueshift.gg/en): structured learning paths for Solana development
    - [Wallets](https://solana.com/wallets): explore Solana wallets to manage your assets
- [Rust docs](https://doc.rust-lang.org/book/title-page.html): this is the language used in developing Solana
