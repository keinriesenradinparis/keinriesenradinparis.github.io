**Emacs :**
- EMACS Keybindings :
  - 光标移动 (Navigation)
    * Ctrl + A：移动到行首 (A = Start of Alphabet)
    * Ctrl + E：移动到行尾 (E = End)
    * Ctrl + P：上移一行 (P = Previous)
    * Ctrl + N：下移一行 (N = Next)
    * Ctrl + B：后退一个字符/向左移动 (B = Back)
    * Ctrl + F：前进一个字符/向右移动 (F = Forward)
  - 文本编辑 (Editing)
    * Ctrl + D：删除光标右侧的字符 (D = Delete)
    * Ctrl + H：删除光标左侧的字符 (退格键)
    * Ctrl + K：剪切从光标位置到行尾的所有文本 (K = Kill)
    * Ctrl + Y：粘贴刚才剪切的文本 (Y = Yank)
    * Ctrl + T：交换光标左右两侧的字母 (T = Transpose)
    * Ctrl + O：在光标后插入换行符，但光标停留在原地
  - 视图控制 (View Control)
    * Ctrl + V：向下滚动页面
    * Ctrl + L：将当前行居中显示（或清屏）
